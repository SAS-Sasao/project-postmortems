# 006: スキル標準化・社内DXのロードマップ不在

- **日付**: 2026-03-08
- **業務種別**: 方法論策定（ID: K08）
- **事実（Fact）**: スキル標準化・社内DXが描く未来像と、そこに至るまでの現実的なスケジュール・ロードマップが存在しない
- **課題（Problem）**: ゴールと計画がないまま個別施策を進めているため、活動の方向性が定まらず、成果の測定もできない状態にある
- **原因（Cause）**: 中長期ビジョンの策定と、それをマイルストーンに分解して計画化するプロセスが実施されていなかった
- **影響度**: 高（ID: H）
- **ステータス**: 未着手（ID: S01）

## 改善アクション

| # | アクション | 期限 | ステータス |
|---|-----------|------|-----------|
| 1 | 5年後の理想像（データドリブン経営、データレイク・DWHフル活用）を描いたビジョンドキュメントを作成する | - | 未着手 |
| 2 | ビジョンから逆算したロードマップ（下記参照）を作成し、半年後・1年後・3年後・5年後のマイルストーンを設定する | - | 未着手 |
| 3 | ロードマップを社内全体に共有し、各マイルストーンの達成状況を四半期ごとにレビューする運用を確立する | - | 未着手 |

### ロードマップの例

| 時期 | マイルストーン | 主な施策 |
|------|-------------|---------|
| 半年後 | 情報基盤の整備 | 情報ソースの統合、SharePoint整理、GitHub権限統一 |
| 1年後 | 標準化ドキュメントの刷新完了 | 全ドキュメントの現代化、テンプレート・チェックリスト整備 |
| 3年後 | データ基盤の構築 | データレイク導入、案件情報のデータ化、分析基盤の構築 |
| 5年後 | データドリブン経営の実現 | DWHフル活用、営業レポート自動化、マーケティング分析の実運用 |

## 次の打ち手（仮説）

- 5年後のビジョンを明確に描き共有することで、個別施策が「何のためにやっているのか」が全社員に伝わり、活動への参加意欲と納得感を高められるのではないか
- マイルストーンを設定し四半期レビューを行うことで、進捗の遅れを早期に検知し、計画の軌道修正を適切なタイミングで行えるのではないか
