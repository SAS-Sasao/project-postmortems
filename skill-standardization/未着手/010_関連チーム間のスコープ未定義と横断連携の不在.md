# 010: 関連チーム間のスコープ未定義と横断連携の不在

- **日付**: 2026-03-09
- **業務種別**: 方法論策定（ID: K08）
- **事実（Fact）**: 社内スキル標準化に関わりのあるチーム（社内スキル標準化活動、SharePoint移行推進活動、AI推進活動、社内DXGr等）が複数存在するが、どのチームがいつまでに何を担当・対応するかのスコープが明確化されていない
- **課題（Problem）**: チーム間のスコープが不明確なため、作業の重複や連携不足が発生し、各チームの活動が非効率になっている
- **原因（Cause）**: 各チームの担当範囲・責任分界点を定義するプロセスがなく、チーム横断で全体を俯瞰しハブとなる担当者も不在だったため、各チームが個別に活動する状態が続いていた
- **影響度**: 高（ID: H）
- **ステータス**: 未着手（ID: S01）

## 改善アクション

| # | アクション | 期限 | ステータス |
|---|-----------|------|-----------|
| 1 | 関連チーム（社内スキル標準化活動、SharePoint移行推進活動、AI推進活動、社内DXGr等）の一覧を作成し、各チームの現在の活動内容・成果物・メンバーを整理する | - | 未着手 |
| 2 | 各チームの担当スコープ（対象領域・責任範囲・期限目標）を定義したスコープ定義書を作成し、関係者間で合意する | - | 未着手 |
| 3 | チーム横断のハブ担当者を数名選定し、定期的な横断ミーティングの運用ルール（頻度・アジェンダ・参加者）を策定する | - | 未着手 |
| 4 | 各チームの活動状況・進捗を一元的に可視化するダッシュボードまたは管理表を作成し、作業重複の早期検知と連携促進の仕組みを整備する | - | 未着手 |

## 次の打ち手（仮説）

- 各チームのスコープを明確化し責任分界点を定義することで、作業重複を解消し各チームが自分の担当範囲に集中できるようになるのではないか
- チーム横断のハブ担当者を立てて定期的な横断ミーティングを実施することで、チーム間のコラボレーションが促進され、活動全体の効率が向上するのではないか
